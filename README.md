## What This Application Does

The system allows a user to:

- Enter patient details (name, age, gender)
- Search diseases with smart suggestions
- Select a disease from a list
- View:
  - Symptoms
  - Prevention
  - Medicines
  - Medical advice
  - Specialist recommendation
  - Emergency signs
- See a basic risk indicator based on age
- Track recent search history

Everything runs locally without internet.
